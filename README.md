## 📌 Sobre o Projeto

A **NEVRA SYSTEMS** é uma landing page moderna desenvolvida com foco em:

- Conversão de clientes
- Presença digital forte
- Visual cyber minimalista
- Performance e responsividade
- Automação e soluções digitais

O projeto representa a identidade visual da Nevra Systems, trazendo um design futurista e estratégico voltado para empresas que precisam fortalecer sua presença online.
